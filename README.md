# About

This project was created as part of my bachelor thesis.
The thesis is written in German, but it can easily be translated.

[Bachelor Thesis Repository](https://github.com/Raphael2b3/bachelorthesis_floyd_rose_tuner?utm_source=chatgpt.com)

Tuning a Floyd Rose guitar can be extremely frustrating.
When you tune one string, the tension of the bridge changes, which also affects all other strings. This often leads to an endless tuning loop where every adjustment throws the entire guitar out of tune again.

This app solves that problem.

The app first performs a one-time calibration process for your specific guitar.
The calibration usually takes around 7–10 minutes and only has to be done once.

After calibration, the app can guide you through the tuning process very quickly and accurately. Instead of guessing, the app tells you exactly how each string needs to be adjusted so that all strings end up perfectly in tune together.

# Getting Started

Run the following commands in the terminal:

```bash
flutter pub get

dart run build_runner watch --force-jit
```

Then start the project on a mobile device or emulator.
